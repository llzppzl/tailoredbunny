# TailoredBunny

让 AI Agent 拥有"性格"的项目。像切换 skill 一样，随时切换 MBTI 人格模式。

## 核心功能

**MBTI 模式切换** - 说任意 MBTI 类型，AI 立即切换对话风格

```
用户: infp
AI:   已切换到 INFP 模式 ☺️

用户: entj
AI:   已切换到 ENTJ 模式
```

## 文件结构

| 文件 | 用途 |
|------|------|
| `skills/mbti-*.md` | 通用 baseline（不变） |
| `memory/customized-*.md` | 你的私人进化版 |

## 进化机制

**Baseline** = 通用风格（不变）
**Customized** = 你的私人调整（会累积）

切换模式时，两者**合并** = 完整的风格

**怎么进化：**
1. 你给反馈（如"太啰嗦了"）
2. AI 更新 `memory/customized-*.md`
3. 下次加载时，你的私人部分增强
